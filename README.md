# file_transfer_system
这是一个基于PyQt5图形界面框架和PuTTY工具集开发的跨平台文件传输系统。项目旨在提供一个用户友好的图形界面，用于通过SSH协议安全地传输文件到远程Linux服务器。

核心功能特性
1. 图形用户界面 (GUI)
使用PyQt5构建的现代化界面
简洁的登录表单：用户名、主机IP、密码输入
文件选择对话框
实时进度显示和状态反馈
2. 安全文件传输
基于PuTTY的PSCP工具进行安全文件传输
支持SSH主机密钥指纹验证
密码加密传输
3. 实时进度监控
多线程架构确保UI响应性
实时解析传输进度信息
可视化进度条显示传输状态
4. 跨平台支持
自动检测操作系统类型（Windows/Linux）
智能路径处理（自动转换路径分隔符）
自动安装必要的依赖工具
技术架构
核心模块
1. putty_script.py - 后端逻辑层
putty_script.py
封装了PuTTY命令行工具的调用
提供connect_transmit函数进行文件传输
支持实时进度回调机制
2. mainwindow.py - 前端界面层
mainwindow.py
主窗口类MainWindow
多线程传输类TransmitThread
信号槽机制实现线程间通信
3. File_transfer_system.py - UI定义
File_transfer_system.py
由Qt Designer生成的界面布局代码
定义了登录表单的基本组件
工作流程
用户输入验证：检查用户名、主机IP、密码是否填写
文件选择：通过对话框选择要传输的本地文件
环境检测：自动检测操作系统并安装必要工具
主机验证：获取远程主机的SSH指纹
路径处理：解析远程目标路径
文件传输：使用PSCP进行安全传输
进度监控：实时显示传输进度和状态
结果反馈：传输完成后显示成功/失败信息
技术亮点
🔧 智能依赖管理
自动检测并安装Scoop包管理器（Windows）
自动安装PuTTY工具集
设置PowerShell执行策略和编码
🚀 多线程架构
使用QThread实现后台传输
信号槽机制确保线程安全
避免UI界面卡顿
📊 实时进度解析
解析PSCP输出的进度信息
提取文件大小、传输速度、预计完成时间
动态更新进度条和状态标签
🔒 安全机制
SSH主机密钥指纹验证
安全的密码传输
错误处理和异常捕获

使用场景
系统管理员：快速部署脚本和配置文件到多台服务器
开发人员：将本地代码同步到开发/测试服务器
运维人员：批量传输日志文件或配置文件
教育用途：学习PyQt5和网络编程的实践项目
扩展性
项目具有良好的模块化设计，易于扩展：

支持添加批量文件传输功能
可集成更多PuTTY工具（如Plink远程命令执行）
支持自定义传输协议和加密方式
可添加文件管理功能（本地/远程文件浏览）
这个项目展示了如何将命令行工具与现代化图形界面相结合，为用户提供既安全又便捷的文件传输解决方案。
